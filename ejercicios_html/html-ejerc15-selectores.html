Tipos de selectores que entran en examen

Clase (class=”.”)

El selector de clase class= "." sirve para cambiar de color un elemento específico, ya sea un título, un párrafo o el fondo.
Para seleccionar todo tipo de elementos con una clase específica, escriba un carácter de punto (.) seguido de la clase valor del atributo.
El selector element.class selecciona los elementos especificados con el valor del atributo de clase especificado.
Para seleccionar solo un tipo de elementos con una clase específica, escriba el elemento name, luego un carácter de punto (.), seguido del valor del atributo de clase
Por ejemplo, quiero que el siguiente título h1 sea de color azul pero el que hay debajo (que también es un h1) lo quiero de otro diferente, para ello realizaremos unos cambios abriendo una etiqueta style y creando dos clases con diferentes colores.

Todos los elementos (*)

El selector universal CSS () selecciona todos los elementos de cualquier tipo.*
El selector universal () también puede seleccionar todos los elementos dentro de otro elemento.*
Este selector también se puede espaciar con nombres cuando se utiliza @namespace.
ns|* - selecciona todos los elementos en el espacio de nombres ns
*|* - selecciona todos los elementos
|* - selecciona todos los elementos sin ningún espacio de nombres declarado
Ejemplo: el documento tiene letra arial.
